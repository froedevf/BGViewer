# BGViewer
Blood Glucose Viewer (either Roku streaming app or hosted web service app)

THIS APPLICATION IS NOT FOR MEDICAL USE.

This application is an unofficial, independent viewer for Nightscout CGM data and/or Dexcom Share data.  It is not approved by the FDA or any regulatory authority and is not intended to be used for medical decisions, treatment or diagnosis.

Always rely on your approved glucose meter and CGM device for medical decisions. Consult your healthcare provider before making any changes to your diabetes management.

This application is to be used at your own risk, without any warranties of any kind.

There are two independent applications in this repository. There is a php web app that can be hosted on most web hosting services that support the latest php implementations. A separate app can be downloaded from the roku store (or sideloaded from this repository, following roku sideloading directions).
